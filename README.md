# GUC Downloader

**GUC Downloader** is a Chrome extension with 1000+ downloads that makes downloading files from the GUC CMS website faster and easier. It automatically renames files with clear, readable names so students can quickly identify content without manually renaming files.

---

## Features

- **Readable File Names:** Automatically formats filenames to a clean, understandable structure.  
- **Customizable Naming Options:**  
  - Remove course codes from filenames  
  - Clean up underscores, parentheses, and other formatting clutter  
  - Strip leading numbers from assignment or content names  
- **Auto-Save Settings:** Changes to preferences are saved automatically.  
- **Easy to Use:** Click the extension icon and download—no extra steps required.  

---

## Example

**Original Filename from CMS:**  
GUC_74_53_6399_2021-03-24T17_08_11.pdf


**Renamed by GUC Downloader:**  
CSEN503 Practice Assignment 3.pdf


---

## Installation

1. Install the extension from the [Chrome Web Store](https://chromewebstore.google.com/detail/guc-downloader/enndgogigccocaklahipianacdgepckm). 
2. Click the GUC Downloader icon in your browser toolbar.  
3. Adjust settings if desired—changes are saved automatically.  
4. Refresh the CMS page to apply your settings.  

---

## How to Use

1. Navigate to a course page on the GUC CMS.  
2. Click the download button for any file.  
3. The file will download with a clean, readable filename according to your settings.  

---

## Notes

- Designed specifically for the German University in Cairo CMS system.  
- Works best with Chrome and Chromium-based browsers.
